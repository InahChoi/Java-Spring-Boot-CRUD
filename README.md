# Java-Spring-Boot-CRUD
---

## 📍 What's inside
---
This project is based on the Spring Boot project and uses these packages
- Gradle Project
- Java 11 JDK
- Spring Boot -v 2.5.7
- Packaging Jar
- Lombok

## 📍 Usage
---
Run the project through the IDE and head out to
http://localhost:8080
